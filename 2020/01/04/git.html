Git基本指令

因為想紀錄一下自學的相關筆記，所以就研究了如何用github上傳靜態網，做個blog，也當作是作品集吧！

由於以往都是使用圖像化介面操作git，這次想學習如何使用cmd指令達成需求。

pwd

print working directory，顯示目前所在的目錄結構。

ls

列出目前的檔案跟資料夾。

cd Desktop/

移動到該目錄。

＊先打前面幾個字再案tab，就會自動幫你補完資料夾的名稱。

clear

清空cmd畫面。

cd ~

跳回家目錄。

如果這個專案你是第一次看到，想要下載到你的電腦裡，請使用 Clone 指令；如果你已經下載回來了，你只是想要更新最新的線上版內容，請使用 Pull（或 Fetch）指令。

git clone https://github.com/你的帳號/你的帳號.github.io.git

git status

觀看資料夾檔案狀態。

git add -A

git commit -a -m “index.html”

-a ALL -m”commit版本訊息”。

git push
